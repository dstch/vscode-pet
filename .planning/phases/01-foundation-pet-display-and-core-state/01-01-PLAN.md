## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| client→webview | Untrusted: webview content from extension, CSP restricts resources |
| extension→webview | Trusted: extension controls HTML/JS served |
| webview→extension | Untrusted input: user interactions via postMessage |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-01-01 | Tampering | WebView HTML | mitigate | CSP restricts all scripts to ${webview.cspSource} |
| T-01-02 | XSS | Pet display | mitigate | Sanitize all user-controlled content, no innerHTML |
| T-01-03 | Info Disclosure | Pet state | accept | State is non-sensitive game data |
| T-01-04 | Denial of Service | WebView panel | mitigate | Memory cleanup on dispose, no infinite loops |
</threat_model>
